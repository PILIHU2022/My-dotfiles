- OS:Arch Linux
- WM:Hyprland

# 安装Hyprland

```
paru -S hyprland-git
```

```
paru -S kitty xdg-desktop-portal-hyprland-git qt5-base qt5-wayland qt6-base qt6-wayland
```

# 安装其他软件

```
sudo pacman -S dunst # 通知
```

```
paru -S waybar-git # 状态栏
```

# 目录中所含的文件：

```
.luarc.json: 用于为LSP添加Hyprland特有的补全以及检查，以便于coding
.stylua.toml: 设置代码格式化选项
```
